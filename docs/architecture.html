Key Insight: Context Isolation
All three agents use the same Qwen 3.5 4B model (same weights, same GGUF). The different behaviors emerge entirely from what context each agent receives and what system prompt frames its task. The Scorer never sees the conversation. The Worker never sees the user. The Archivist never sees raw scores. Isolation prevents context bleed between roles.

Key Insight: Vocabulary Priming
The Archivist asks "what's on your tapes?" not because it's curious — it uses the answer to generate a vocabulary list (via the Worker agent) that gets injected into Whisper's decoder as an initial_prompt. This biases Whisper toward domain-specific words it would otherwise mangle. "Om Namah Shivaya" instead of "oh no my shivaya." The user thinks they're having a conversation. The system is optimizing a speech recognition pipeline.

Key Insight: Runtime Steering
The Scorer rates labels 1-10 and states the human consequence of errors: "your label forced the user to rename files manually." This isn't anthropomorphizing — it's applying targeted pressure that these agentic-trained models respond to. A score of 2/10 with a concrete consequence adjusts the next output more reliably than a paragraph of instructions. Too much negative pressure breaks small models (they collapse into apologizing). One consequence statement is the right calibration.

Data Flow — Single Tape

1. User → Archivist
"I have kirtan tapes from our ashram. Anandamayi Ma is on most of them."
2. Archivist → Worker (vocabulary)
Worker reads skill briefing, generates: "Anandamayi Ma, Om Namah Shivaya, satsang, kirtan, bhajan, pranayama..."
3. Archivist → User
"Got it — I've tuned my ears for spiritual recordings. Load the first tape when ready."
4. start_capture → encode → transcribe (with vocabulary priming)
ffmpeg records → x264 encodes → Whisper transcribes with "Anandamayi Ma, Om Namah Shivaya..." as initial_prompt
5. Worker → label_tape
Reads transcript, generates: {"title": "Kirtan with Anandamayi Ma — Om Namah Shivaya", "tags": ["kirtan", "ashram"]}
6. Scorer → score_label
Score: 8/10. "Tag '1990s' not in transcript — minor hallucination." No consequence (score ≥ 7).
7. save_metadata + save_memory
Metadata JSON saved alongside MP4. Vocabulary appended to memory for next session.
